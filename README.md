# Secure-healthcare-Networking-System-using-Cisco-packet-tracer
A Cisco Packet Tracer simulation of a HIPAA-compliant healthcare network featuring VLAN segmentation, ASA firewall, IPSec VPN, RADIUS authentication, and IDS/IPS monitoring to secure clinical, administrative, and IoT systems across a multi-department hospital environment.
